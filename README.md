## 📝 Sobre

O Projeto Reciclar foi desenvolvido como parte da disciplina Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas. O objetivo desta disciplina é realizar um projeto por semestre para integrar e aplicar os conhecimentos adquiridos ao longo do curso. O Projeto Reciclar é um site desenvolvido apenas com front-end, utilizando HTML, CSS e JavaScript.

## 🔨 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

Contribuições são bem-vindas! Se você quiser contribuir para este projeto, siga estas etapas:

1. Faça um fork do projeto
2. Crie um novo branch (`git checkout -b feature/nova-funcionalidade`)
3. Faça commit das suas alterações (`git commit -am 'Adiciona nova funcionalidade'`)
4. Faça push do branch (`git push origin feature/nova-funcionalidade`)
